================================================================================
                           DESIGN PHILOSOPHY
================================================================================

CHARACTER CONTRAST:
  Generalissimo = Tank: High health, strong hits, methodical pressure
  Demagogue = Agile: Lower health, fast startup, constant poking

PLAYSTYLE DIFFERENCES:
  Generalissimo: "Win through superior strength"
    • Slow, steady damage accumulation
    • Forgiving to play (large health pool)
    • Beginner-friendly
    • Strong single hits = KO threat

  Demagogue: "Win through superior speed"
    • Fast, constant pressure
    • Execution-dependent (need good spacing)
    • Intermediate difficulty
    • Chip damage accumulation = grinding wins

VISUAL IDENTITY:
  Generalissimo: Red/Brown suit, heavy authoritarian presence
  Demagogue: Green suit, light theatrical presence
  Clear visual distinction for quick recognition

================================================================================
                          QUICK START GUIDE
================================================================================

To Use Demagogue in Battle:

1. LOAD CHARACTER DATA:
   var demagogue_data = load("res://game/resources/characters/demagogue_1.json")

2. CREATE CHARACTER INSTANCE:
   var demagogue = preload("res://game/scenes/characters/demagogue_1.tscn").instantiate()
   add_child(demagogue)

3. ACCESS CHARACTER STATS:
   var stats = demagogue.get_stats()
   # Returns: { health: 85, speed: 220, weight: 0.8, ... }

4. EXECUTE A MOVE:
   var move = demagogue.execute_move("light_attack")
   # Returns: { move_id, name, damage, startup_frames, active_frames, recovery_frames, knockback, hitbox }

5. IMPLEMENT ANIMATIONS:
   See: game/scenes/characters/demagogue_1_animations.md
   Required: 8 animation states in AnimationPlayer
